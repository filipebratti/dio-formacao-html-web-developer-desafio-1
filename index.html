Hierarquia de Títulos

Os títulos em HTML organizam o conteúdo em diferentes níveis de importância.

Este é um <h1>

O título principal da página, deve aparecer apenas uma vez.

Este é um <h2>

Subtítulo principal, agrupa seções maiores.

Este é um <h3>

Terceiro nível, para subdivisões.

Este é um <h4>

Este é um <h5>
Este é um <h6>

Títulos menos importantes, usados raramente.

Formatação de Texto

Ênfase com <strong>

Use strong para indicar importância semântica.

Itálico com <i>

Use itálico para citações, nomes científicos e termos estrangeiros.

Sublinhado com <u>

Use sublinhado com cuidado, não confundir com links.

Destaque com <mark>

Use mark para destacar trechos importantes do texto.

Texto Pequeno com <small>

Este é texto normal. Este é um texto pequeno.

Deletado com <del>

O preço era R$ 100,00 agora custa R$ 79,90.

Combinação: Texto importante em itálico com destaque e sublinhado

Links e Âncoras

Links Externos

A tag <a> cria hiperlinks para outras páginas.

Visite DIO - Digital Innovation One para mais cursos.

Confira a documentação completa de HTML na Mozilla Developer Network.

Links Internos

Você pode criar links para seções da mesma página.

Atributos Comuns

  • href - URL de destino
  • target="_blank" - Abre em nova aba
  • title - Texto ao passar o mouse
  • rel - Relacionamento com o destino

Citações

<blockquote>

Use para citações longas e importantes:

"O melhor lugar para começar é onde você está agora. Não espere pelas condições perfeitas; elas nunca vêm." - David C. Thomas
"Web design é como design gráfico, com a restrição adicionada de usar navegadores. Mas não é um limite, é uma oportunidade." - Desenvolvedor Experiente

<hr> - Separador Horizontal

Use para separar seções de conteúdo não relacionadas.

A linha acima é criada com a tag <hr>.

Tags Avançadas

<abbr> - Abreviações

A HTML trabalha junto com CSS e JS para criar experiências web modernas.

<sub> e <sup>

A fórmula da água é H2O.
A velocidade da luz é c ≈ 3×108 m/s.
Referência[1] em artigos acadêmicos.

<mark> - Marcação Semântica

Use <mark> para destacar conteúdo relevante para o usuário, não apenas para estilo visual.

<font> - Tag Legada

<font color="red"> é obsoleta em HTML5

Prefira usar CSS para colorir texto. Exemplo:

<span style="color: #ff8c00;">Texto colorido</span>

Combinando Tags

Resumo das Tags Utilizadas

Estrutura: <h1> até <h6>, <p>, <hr>
Formatação: <mark>, <small>, <i>, <u>, <strong>, <del>
Listas: <ol>, <ul>, <li>
Links: <a>
Avançado: <blockquote>, <abbr>, <sub>, <sup>, <font>
